Flow 03 / 03

Emergency (SOS) Flow

Screens 10–12 · SOS Active → Contact View → Emergency Resolved

User journey: Hold red SOS button for 3 seconds → Escalation countdown begins (2:45) → Contacts alerted with live GPS → User cancels OR timer expires → Contacts see location on red-tinted map.
Safety design: 3-second deliberate hold prevents pocket-bump false alarms. 2:45 window allows cancellation. Shake-to-SOS hint (toast only — full activation still requires hold).

#EF4444

#10B981
Safe state · confirmed arrival, "I'm safe"
Slate
#94A3B8
Muted text · labels, secondary info
Helvetica (Arial) — Title
16pt · Bold · Primary headings, screen titles
Helvetica (Arial) — Body
9pt · Regular · Body copy, descriptions, labels
HELVETICA — CAPTION / LABEL
6pt · Uppercase · Overlines, metadata, status labels
Dark-first design
Navy background designed for night-safety use. High contrast ensures visibility in dim or stressful conditions.
Minimum touch targets
SOS button ≥ 48dp. All interactive elements sized for one-handed operation under stress.
Prevention over emergency
Safe Arrival is the daily habit driver. SOS is the safety net. UI hierarchy reflects this intentionally.
Network resilience
Leaflet embedded inline — zero CDN dependency. Tiles fail gracefully to grey. Tested on MTN 3G.
