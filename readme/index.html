🎯 The Problem

Nigerians — especially women, professionals working late, and students — face daily safety friction. There's no fast, low-friction way to tell trusted contacts "I'm on my way home" or "I need help" without unlocking your phone, finding the right person, and typing. Existing options (WhatsApp share-location, calling a friend) are slow, social, and don't scale to emergencies.

Target user: Ada, 28, project manager in Lagos. Takes Bolt rides home after late meetings. Has two trusted contacts (sister + close friend).

🚀 Demo Cheatsheet (for judges)

Open the  live app on a phone, then:

Action What to expect
Tap Get started Walk through onboarding
OTP code Use 8484 (any other code triggers the failure path)
Home screen Hold the red SOS button for 3 seconds → emergency activates
Start safe arrival card Tap Allow on the GPS banner → real OpenStreetMap loads with your live location
Bottom demo nav Jump instantly between flows: 🔑 Onboard · 🏠 Safe Arrival · 🚨 SOS · ⚠️ Failures

Three failure paths to demo: wrong OTP code · location permission denied · trip ETA overdue Two delighters: confetti on safe arrival · shake-to-SOS hint

🛠️ Tech Highlights

  • Single-file HTML app (~210 KB) — deploy to any static host
  • Real OpenStreetMap via Leaflet — embedded inline, zero CDN dependencies
  • Works on the worst Nigerian networks: library is self-hosted, tiles load progressively, error fallbacks for failed tiles
  • Real GPS via navigator.geolocation + on-screen permission flow
  • Native haptics for SOS (Android navigator.vibrate)
  • Shake detection via DeviceMotion API
  • No build step. No dependencies. No backend (yet — V2 will wire to Supabase like SafeSignal).
